# Bitcoin Price & Trading Analysis Dashboard

## Overview

This project is an interactive dashboard analyzing Bitcoin price trends and trading volume over time.

## Features

* Price movement over time
* Monthly price vs trading volume analysis
* Relationship between trading volume and price
* Key metrics (highest, lowest, latest price)

## Tools Used

* Power BI / Tableau / (your tool here)
* Data visualization techniques
* Financial data analysis

## Key Insights

* Bitcoin price shows fluctuations over time with periods of decline
* Trading volume varies monthly and may influence price trends
* Weak/moderate relationship observed between volume and price
